# 🔗 Encurtador de URLs

API REST desenvolvida em **Java 25 + Spring Boot** para criação de URLs encurtadas com suporte a:

- alias customizado
- data de expiração
- contagem de cliques
- internacionalização (PT / EN / ES / FR)
- validação de URL
- tratamento global de erros

---

# 📦 Tecnologias utilizadas

- Java 25
- Spring Boot 3
- Maven
- Jackson
- Spring Validation
- PowerShell (scripts de teste)
